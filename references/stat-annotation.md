# 统计标注与图注义务

使用场景：图中包含误差棒、阴影区间、模型拟合、p 值、显著性括号、星号或样本量时阅读本文件。

## 不得编造统计量

不要凭空生成 p 值、n、置信区间、SD、SEM、检验方法或多重校正。只有在用户提供结果，或可以从原始数据按明确方法计算时，才允许写入图和审计报告。

## 必须在图注或 provenance 中交代

- 误差类型：SD、SEM、95% CI、IQR、bootstrap CI、credible interval 等。
- 样本量 `n`，并说明 n 代表生物重复、技术重复、受试者、试验还是测量点。
- 如果显示显著性：统计检验或模型。
- 多重比较校正方法。
- 显著性符号定义，例如 `* p<0.05`。

## 图型选择约束

- 小样本：显示原始点；不要用无原始点的均值柱掩盖分布。
- 配对/重复测量：显示配对连线或 within-subject delta。
- 偏态正值数据：考虑 log 轴或分布式摘要；不要用均值柱隐藏偏态。
- 回归拟合：只有模型假设合理时才画拟合线和 CI。
- 热图：必须有 colorbar label 和单位；发散色图必须以有意义的零点/参考点为中心。

## 显著性标注

- 括号和星号不能遮挡数据或被裁切。
- 比较很多时不要堆叠大量括号，改用紧凑字母、effect size、补充表格或分面。
- 优先展示 effect size 和 interval，不要只讲星号。
